How to Use the Tool

Here's how to use the Postcode Clustering Tool:

  1. Upload your CSV file containing postcodes
  2. Select the number of groups you want to create
  3. Click "Generate Map" to see your clustered results
  4. Use "Download Clusters" to get your labeled data

The tool will automatically group your postcodes into the specified number of clusters based on geographical proximity.

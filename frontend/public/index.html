How to Use the Tool

Here's how this Machine Learning-based Postcode Clustering Tool works:

  1. Upload your CSV file containing postcodes organized in a single column with no headers
  2. Select the number of clusters you want to create
  3. Click "Postcode Map" to see on the map the clustered postcodes by color
  4. Use "Download Clusters" to get an excel file with the list of each postcode and its cluster number

The tool will automatically group your postcodes into the specified number of clusters based on a machine learning algorithm that uses geographical proximity.
